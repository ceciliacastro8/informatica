Cursos de Informática

Programación Web

En este curso aprenderás desde cero a construir páginas web utilizando HTML para la estructura, CSS para el diseño visual y JavaScript para añadir interactividad. Durante 8 semanas conocerás cómo crear formularios, menús interactivos y páginas adaptables a móviles, además de publicar tus propios sitios en línea. Es ideal para principiantes, estudiantes o cualquier persona interesada en iniciar su camino como desarrollador web.

Ciberseguridad

es el conjunto de prácticas y tecnologías destinadas a proteger sistemas informáticos, redes y datos frente a accesos no autorizados, ataques o daños. En este curso aprenderás los fundamentos esenciales para detectar amenazas, prevenir hackeos, proteger contraseñas, identificar virus y navegar de forma segura..

Bases de Datos

son sistemas organizados que permiten almacenar, gestionar y recuperar grandes cantidades de información de forma eficiente y segura.
